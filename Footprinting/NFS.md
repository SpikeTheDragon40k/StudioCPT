## Mounting NFS Share
```bash
xxxxxxxxxxxxx@htb[/htb]$ mkdir target-NFS
xxxxxxxxxxxxx@htb[/htb]$ sudo mount -t nfs 10.129.14.128:/ ./target-NFS/ -o nolock
xxxxxxxxxxxxx@htb[/htb]$ cd target-NFS
xxxxxxxxxxxxx@htb[/htb]$ tree .

.
└── mnt
    └── nfs
        ├── id_rsa
        ├── id_rsa.pub
        └── nfs.share

```

## Unmounting
```bash
sudo umount ./target-NFS
```
